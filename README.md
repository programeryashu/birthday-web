# 🎂 Premium Birthday Surprise App

A modern, mobile-first birthday surprise application built with React, Tailwind CSS, and Framer Motion.

## ✨ Features
- **Remote Cloud Sync**: Powered by Supabase to keep your configuration synced across all devices.
- **Admin Panel**: Double-click the 🎁 icon to customize names, letters, and themes.
- **Premium Animations**: Smooth transitions using Framer Motion.
- **Responsive Design**: Looks beautiful on phones, tablets, and desktops.

## 🚀 Deployment
This app is optimized for Netlify/Vercel.

### Environment Variables
Ensure the following variables are set in your hosting provider:
- `VITE_SUPABASE_URL`: Your Supabase Project URL.
- `VITE_SUPABASE_ANON_KEY`: Your Supabase Anon/Public Key.

## 🛠️ Tech Stack
- React
- Tailwind CSS
- Framer Motion
- Supabase
